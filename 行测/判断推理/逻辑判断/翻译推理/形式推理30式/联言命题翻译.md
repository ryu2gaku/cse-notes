# 联言命题翻译

```
定义：联言命题是反映事物的若干种情况或者性质同时存在的命题。
符号：Ａ∧Ｂ
关联词：
1. 并列关系：并且、且、和、都、即……又……、一边……一边……
2. 递进关系：而且、甚至、还……
3. 转折关系：但是、然而、却……
4. 标点符号：逗号、顿号、分号
```

【例 1】小王和小李都上岸了。以下哪项逻辑含义与上述含义不一致。  
A. 小王且小李都上岸了  
B. 不仅小王上岸了，而且小李也上岸了  
C. 虽然小王上岸了，但是小李也上岸了  
D. 要么小王上岸了，要么小李上岸了

```
答案：D
```

【例 2】东边下雨，西边彩虹。以下哪项逻辑含义与上述含义不一致。  
A. 不仅东边下雨，而且西边彩虹  
B. 虽然东边下雨，但是西边彩虹  
C. 只要东边下雨，就西边彩虹  
D. 东边下雨且西边彩虹

```
答案：C
```
